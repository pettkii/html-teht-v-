Javascript

var, let ja const

let ja const ovat uudempia tapoja alustaa muuttujia ja niillä on eri ominaisuuksia etenkin verrattuna vanhempaan var tapaan.

var tavalla alustetulla muuttujalla on global scope eli blockin sisällä alustettua muuttujaa voidaan käyttää blockin ulkopuolella, kun taas let ja const muuttujilla voi olla block scope eli blockin sisällä alustettuna niitä ei voi käyttää blockin ulkopuolella.

let ja const muuttujia ei voi uudelleenalustaa(redeclare), kun taas var muuttujia voi uudelleenalustaa.

var ja let muuttujia voi uudelleenmäärittää(reassign) kun taas const muuttujia ei voi uudelleenmäärittää
